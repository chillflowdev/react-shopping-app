# Shopping App

## 📌 概要
Context API と `useReducer` を利用し、状態管理のベストプラクティスを学ぶために実装したショッピングアプリです。カート操作や合計金額計算などのロジックを含みます。

## 🛠️ 技術スタック
- React
- Context API
- useReducer

## 🎯 主な機能
- 商品一覧表示
- カートへの追加・削除
- 数量変更および合計金額の計算

## 💡 学習ポイント
- props drilling 解消のための Context 導入
- `useReducer` による明確な状態遷移管理
- Reducer 内でのクロージャを活用した安全な状態管理
